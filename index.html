sobre nós

Nós somos uma Empresa Júnior formada pelos alunos dos cursos de Engenharia de Computação, Engenharia de Software e Análise e Desenvolvimento de Sistemas da UTFPR-CP.

Temos como missão construir soluções sob medida e inovadoras ao cliente, tudo isso com preços mais acessíveis e com ótima qualidade, sendo que as nossas soluções vão desde sites até aplicativos e sistemas mais complexos.

Além disso, visamos proporcionar uma vivência empresarial aos estudantes da área de computação da universidade, desenvolvendo tanto suas habilidades técnicas quanto habilidades comportamentais.

ETAPAS DO

PROCESSO SELETIVO

Inscrições

A primeira etapa é a que aproxima e faz o primeiro contato entre a empresa e o candidato. Durante o período de inscrição, o candidato deve buscar conhecer mais sobre a Unect e escolher qual a diretoria mais se identifica através do formulário de inscrição.

Entrevista

A etapa de entrevistas é o momento de conhecer melhor o candidato, levantando informações de perfil e analisando seu fit com a empresa.

Dinamicas

Após serem aprovados nas entrevistas, os candidatos passam por duas dinâmicas, a geral e a específica.
A primeira possibilita a análise do candidato enquanto membro de uma empresa. Já a segunda, é uma dinâmica específica da diretoria designada ao mesmo.

Treinamento

Na última etapa, a lista dos candidatos aprovados é divulgada e os mesmos são introduzidos à empresa como trainees, iniciando uma trilha de treinamento de 2 meses.

Nossa Stack

HTML

É uma Linguagem de Marcação de Hipertexto utilizada na construção de páginas na Web.

CSS

É um mecanismo para adicionar estilo a um documento Web. Utilizado para fazer o design das páginas na Web.

JAVASCRIPT

É uma linguagem de scripting dinâmica utilizada para manipular comportamentos em uma página Web.

REACT JS

É uma biblioteca JavaScript de código aberto utilizada para criar interfaces de usuário.

NODE

É uma plataforma criada pelo Google que permite a execução da linguagem javascript do lado do servidor.

REACT NATIVE

É uma biblioteca Javascript criada pelo Facebook e utilizada para desenvolver aplicativos para Android e iOS.
